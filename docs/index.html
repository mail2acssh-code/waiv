How it works

Three steps, no friction.

All processing happens locally using MediaPipe hand landmarks. Nothing leaves your Mac.

01

Webcam reads 21 landmarks

MediaPipe maps your hand into 21 skeletal points in real time — pure geometry, no ML API, no network call.

02

Hold for 2 seconds to confirm

Waiv requires a steady hold before firing, so accidental gestures never trigger. A ring fills as you hold.

03

Action runs, HUD confirms

A frosted-glass tile slides up to confirm what fired. Then it's back to watching — no app switching needed.

Plugin system

Add gestures for any app.

Drop a .py file into plugins/. Waiv loads it automatically at startup. No registration, no config file.

  • Your plugin overrides defaults only when is_active() returns true
  • Falls back to default media controls the moment your app quits
  • Zoom, Figma, Final Cut, Slack — anything AppleScript can control
# plugins/zoom.py
import subprocess

def is_active():
    r = subprocess.run(
        ["osascript", "-e",
         '(name of processes) contains "zoom.us"'],
        capture_output=True, text=True
    )
    return r.stdout.strip() == "true"

GESTURE_ACTIONS = {
    "open_palm": toggle_video,  # ✋ camera
    "index_up":  toggle_mic,    # ☝️  mute
}
For developers

Clone and run in 60 seconds.

One script handles everything. Contribute a plugin and open a PR.

1

Clone

Git clone the repo and cd in.

2

Run install.sh

Creates a venv, downloads the hand-landmark model, installs the LaunchAgent.

3

Add a plugin

Drop a .py file in plugins/ to control any app.

# clone
$ git clone https://github.com/mail2acssh-code/waiv waiv
$ cd waiv

# set up venv · download model · start agent
$ bash install.sh
