How it works

Three steps, zero friction.

All gesture detection runs locally using MediaPipe. Nothing leaves your machine.

📷

Camera reads your hand

MediaPipe tracks 21 hand landmarks in real time using pure geometry — no ML API, no network calls.

✋

Gesture is recognised

Hold any gesture steady for 2 seconds. Waiv debounces false positives so accidental triggers don't happen.

⚡

Action fires instantly

A frosted-glass HUD confirms what fired. Volume, playback, mic, lock screen — or your own plugin.

Plugin System

Add gestures for
any app.

Drop a .py file into plugins/. Waiv auto-loads it at startup — no registration, no config.

  • Override any default gesture when your app is active
  • Falls back to defaults the moment your app quits
  • No core code changes — just drop the file and restart
# plugins/zoom.py
import subprocess

def is_active():
    # Only activates while Zoom is running
    r = subprocess.run(
        ["osascript", "-e",
         '(name of processes) contains "zoom.us"'],
        capture_output=True, text=True
    )
    return r.stdout.strip() == "true"

GESTURE_ACTIONS = {
    "open_palm": toggle_video,  # ✋ camera on/off
    "index_up":  toggle_mic,    # ☝️  mute/unmute
}
